Demo: OExchange Facebook Wrapper

Get here by accident? Check out OExchange.org.

This site demonstrates OExchange support on top of Facebook. It provides:

  • An OExchange-compatible Target endpoint that wraps the production Facebook endpoint
  • An OExchange Target XRD file that describes the endpoint
  • A Host-Meta that allows you to discover the service
  • A <link> tag in this page that references the Target XRD

You can perform OExchange-Discovery on this host. For example, try it out with the Discovery Test Harness.

Note: This site is NOT provided by Facebook, it is a demo from the OExchange.org development community.
