OExchange Facebook Wrapper

This is a website that demonstrates what it would be like to interact with Facebook using OExchange. This site provides:

  • An OExchange-compatible Target endpoint (at http://oexchange-facebook.appspot.com/offer)
  • An OExchange Target XRD descriptor describing the endpoint
  • A host-meta XRD file for performing host discovery
  • <link> tags in the HEAD of this page that reference the Target XRD

In place of http://facebook.com, you can use http://oexchange-facebook.appspot.com for OExchange discovery. For more information, please visit http://oexchange.org
